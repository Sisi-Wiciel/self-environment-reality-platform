# 自我与环境真实感行为测量平台 PRD

## 项目名称
**自我与环境真实感行为测量平台**  
**Self and Environmental Reality Assessment Platform**

---

## 1. 产品背景

人格解体-现实解体障碍（DPDR）的核心体验包括：
- 自我不真实感
- 环境不真实感
- 身体疏离感
- 主观在场感下降
- 自我—环境边界不稳定

这些体验高度主观、难以客观量化，也难以与 EEG / MRI / 蛋白组等生物指标精确对接。当前多数研究仍依赖量表，而缺乏一个同时满足以下要求的行为平台：

1. 能测“自我感”
2. 能测“环境真实感”
3. 能量化“自我—环境耦合”
4. 能输出结构化指标
5. 能为 EEG / fMRI 联动研究提供标准任务接口

因此，本项目拟开发一个面向 DPDR 及相关研究的行为测量平台，用于量化个体在“自我真实感—环境真实感—主观在场感”维度上的异常特征。

---

## 2. 产品目标

### 2.1 总目标
构建一个用于量化“自我与环境真实感”的行为任务平台，为 DPDR 的：
- 机制研究
- 行为表型提取
- 多模态联合分析
- 软著/专利布局
- 临床转化储备

提供稳定、标准化的数字工具。

### 2.2 分阶段目标

#### 阶段 1：MVP 版本
完成一个可运行的纯行为平台，支持：
- 基础信息录入
- 主观量表录入
- 3 个核心任务
- 行为结果导出
- 基础结果报告

#### 阶段 2：科研工具版本
增加：
- 研究者后台
- 条件配置
- 任务日志
- 批量导出
- 质量控制模块

#### 阶段 3：多模态接口版本
增加：
- EEG marker 输出
- fMRI event 文件导出
- 行为—脑信号对接接口

#### 阶段 4：临床转化版本
增加：
- 个体画像
- 风险分层
- 分型建议（研究版）
- 疗效评估接口

### 7.3 核心行为任务模块
平台 MVP 需包含 3 个核心任务。

#### 任务 1：自我真实感判断任务
**目标**：量化个体的自我感稳定性与自我异化程度。

**刺激形式**：
- 自我相关语句
- 去自我化语句
- 中性语句

**被试任务**：
- 真实性评分（1–7）
- 或是否符合判断 + 自信度评分

**主要指标**：
- 自我真实感指数
- 自我异化指数
- 自我判断波动性
- 平均反应时
- 自信度

#### 任务 2：环境真实感判断任务
**目标**：量化环境是否“像真的”的主观体验。

**刺激形式**：
- 环境图片
- 环境描述语句
- 后续可扩展短视频

**主要指标**：
- 环境真实感指数
- 环境疏离指数
- 熟悉度—真实感偏差
- 反应时

#### 任务 3：自我—环境匹配任务
**目标**：量化自我状态与环境状态在主观体验中的协调性。

**刺激形式**：
- 自我状态描述
- 环境状态描述

**主要指标**：
- 自我—环境协调指数
- 自我—环境失配敏感度
- 匹配反应时
- 评分偏差

---

### 7.4 结果计算模块
#### MVP 输出指标
**主指标**：
1. 自我真实感指数

4. 主观在场感指数

**辅助指标**：
1. 平均反应时

4. 自信度偏差

## 13. 风险与注意事项

1. 不要一开始做太多任务
2. 不要让任务概念太抽象，导致被试难理解
3. 需要留意疲劳效应
4. 任务语句要经过预实验筛选
5. 行为指标必须可重复、可解释、能和量表/脑信号挂钩

---

## 14. 一句话产品定义

> 这是一个用于量化自我真实感、环境真实感和自我—环境协调性的行为任务平台，服务于 DPDR 的机制研究、多模态联合分析与未来临床转化。
