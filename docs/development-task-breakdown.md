# 开发任务拆解清单

---

## 0. 项目目标

将平台开发拆分成可执行任务，使项目能从 PRD 平稳进入：
- 原型设计
- MVP 开发
- 数据采集
- EEG/fMRI 接口预留
- 后续软著/专利/论文支持
