# DPDR 自我与现实感任务电池框架图
## DPDR Selfhood and Reality Battery Framework

更新时间：2026-03-12
项目：Self and Environmental Reality Assessment Platform

---

## 一、总目标

构建一套面向 DPDR（人格解体-现实解体障碍）的**自我与现实感任务电池（battery）**，用于系统测量以下核心维度：

1. 自我真实感（self-reality）
2. 环境真实感（environmental reality）
3. 身体归属感 / 身体在场感（bodily ownership / embodied presence）
4. 自我—环境耦合（self-environment coupling）
5. 自我边界稳定性（self-boundary stability）

该任务电池未来需服务于：
- 行为学研究
- EEG / fMRI 联动研究
- 多模态生物标志物整合
- 软著 / 专利 / 平台化转化
- 临床辅助评估与分型储备

## 三、五大核心维度与任务对应

### 维度 1：自我真实感（Self-reality）
**核心问题**：
- 我是否感觉自己是真实存在的？
- 我是否像“我是我”？
- 我的主观体验是否连续、稳定、属于自己？

**建议核心任务**：
#### 任务 1A：自我真实感判断任务
- 刺激：自我相关陈述 / 去自我化陈述 / 中性陈述
- 反应：1–7 分评分，或二分类 + 置信度评分
- 输出：
  - 自我真实感指数
  - 自我异化指数
  - 判断波动性
  - 平均反应时

**意义**：
这是 battery 的中轴任务，用于捕捉“我是不是我”的稳定性。

---

### 维度 2：环境真实感（Environmental reality）
**核心问题**：
- 外部世界是否像真的？
- 环境是否有疏离感、隔膜感、梦样感？

**建议核心任务**：
#### 任务 2A：环境真实感判断任务
- 刺激：环境场景图片 / 场景语句 / 后续可扩展视频
- 反应：真实感评分、疏离感评分、在场感评分
- 输出：
  - 环境真实感指数
  - 环境疏离指数
  - 熟悉度—真实感偏差
  - 平均反应时

**意义**：
该模块是 DPDR 非常重要但文献中仍相对空白的方向，具有较强创新空间。

---

### 维度 3：身体归属感 / 身体在场感（Bodily ownership / embodied presence）
**核心问题**：
- 身体是不是自己的？
- 是否“像活在身体外面”？
- 身体感觉是否淡化、遥远？

**建议核心任务**：
#### 任务 3A：身体在场感任务
- 刺激：身体状态语句 / 身体聚焦任务 / 简单身体注意操控
- 反应：身体归属感评分、在场感评分、身体疏离评分
- 输出：
  - 身体归属感指数
  - 身体疏离指数
  - 身体在场感指数

**意义**：
该维度与 DPDR 的身体脱离、内感受异常高度相关，也最适合后续接针刺和免疫—脑环路主线。

---

### 维度 4：自我—环境耦合（Self-environment coupling）
**核心问题**：
- 自己和世界是不是协调的？
- 自我状态和环境状态能否整合？

**建议核心任务**：
#### 任务 4A：自我—环境匹配任务
- 刺激：同时呈现自我状态描述 + 环境状态描述
- 反应：匹配度评分 / 协调度评分 / 是否像自己体验
- 输出：
  - 自我—环境耦合指数
  - 失配敏感度
  - 冲突条件反应时

**意义**：
这是最能体现你理论特色的维度，能把“自我不真实”和“环境不真实”放到同一个结构里测量。

---

### 维度 5：自我边界稳定性（Self-boundary stability）
**核心问题**：
- 自己和他人、自己和世界的边界是否稳定？
- 是否容易出现自我模糊、边界松动？

**建议核心任务**：
#### 任务 5A：自我—他人边界任务
- 刺激：自我脸 / 他人脸 / morphing 刺激 / 自我-他人描述语句
- 反应：更像自己还是他人、边界清晰度评分
- 输出：
  - 边界稳定性指数
  - 自我-他人模糊化偏差
  - 边界判断反应时

**意义**：
这一维度更偏“自我边界”本身，对未来机制升级与专利布局都很重要。

## 四、Battery 输出结构设计

### 1. 分域分数（Domain Scores）
- Self-Reality Score
- Environmental Reality Score
- Bodily Ownership Score
- Self-Environment Coupling Score
- Boundary Stability Score

### 2. 全局总分（Global Score）
- Global Selfhood-Reality Disturbance Score

### 3. 行为辅助指标
- Mean RT
- Conflict RT
- Response Variability
- Confidence Bias
- Task Completion Quality

## 五、平台分阶段版本设计

### Version 1：MVP 平台
包含：
- 任务 1A：自我真实感判断任务
- 任务 2A：环境真实感判断任务
- 任务 4A：自我—环境匹配任务
- 基础问卷/量表页
- CSV / JSON 导出

### Version 2：扩展版任务电池
增加：
- 任务 3A：身体在场感任务
- 任务 5A：自我—他人边界任务
- 研究者后台
- 质量控制模块

### Version 3：标准化 Battery 版本
增加：
- 结构效度检验
- 重测信度
- 常模构建
- 分域解释手册
- 总分报告与解释规则

## 六、与 EEG / fMRI 联动关系

### EEG 联动建议
- 任务 1A：自我判断相关 ERP / time-frequency
- 任务 2A：环境真实感判断中的频段与连接变化
- 任务 4A：匹配/失配条件下的冲突加工和 gamma/theta 特征
- 任务 5A：自我—他人边界加工与自我表征动态

### fMRI 联动建议
- 任务 1A：自我相关加工网络
- 任务 2A：环境真实感与场景加工网络
- 任务 4A：自我—环境匹配中的 insula–mPFC 耦合
- 任务 5A：自我边界与面孔/身份识别相关网络

## 七、Battery 的临床与转化价值

### 科研价值
- 为 DPDR 任务范式提供结构化框架
- 便于和 EEG / MRI / 蛋白组联动
- 可发展为方法学论文与标准化测验研究

### 资产价值
- 软件著作权：任务平台软件
- 专利：任务结构、评分方法、行为—脑信号联合评估方法

### 转化价值
- 门诊辅助评估工具储备
- 分型与疗效预测数据入口
- 针刺干预评估与随访支持工具

---

## 八、一句话总结

> DPDR 自我与现实感任务电池应围绕“自我真实感、环境真实感、身体归属感、自我—环境耦合、自我边界稳定性”五大域来组织，每个域配置至少一个核心任务，逐步从平台原型升级为标准化 battery，并最终服务于多模态机制研究与临床转化。
